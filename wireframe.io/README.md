# Wireframe Output

This folder contains a separate wireframe prototype for a high-end Caribbean tropical beach company.

## Files

- `generate_wireframe.py` : Python script that generates `wireframe.png`.
- `wireframe.png` : Generated wireframe image output.
- `requirements.txt` : Python dependency for generating the PNG.

## Usage

Install dependencies and run the script:

```bash
python -m pip install -r requirements.txt
python generate_wireframe.py
```
